Instrucciones de uso:
En la página del repositorio, haz clic en el botón verde "<> Code".
Copia la URL que aparece en la pestaña HTTPS (se ve algo como https://github.com/usuario/proyecto.git).
Abre la Terminal (Mac/Linux) o Git Bash/CMD (Windows) en tu computadora.
Navega hasta la carpeta donde quieras guardar el proyecto usando el comando cd.
Escribe el siguiente comando y presiona Enter:
git clone https://github.com/usuario/proyecto.git

 Inicio y Carga
Al ejecutar el programa, el sistema buscará automáticamente en la carpeta datos/ cualquier arte guardado previamente. Si es tu primera vez, ¡la galería empezará vacía y lista para ser llenada!
Navegación por el Menú
Verás un menú principal interactivo con las siguientes opciones:
Figuras Geométricas: Elige entre Triángulo, Cuadrado o Pirámide. Deberás ingresar el tamaño (altura o lado).
Texto Artístico: Escribe tu nombre para un Banner o elige un estilo de Marco decorativo.
Animaciones: Disfruta de la barra de progreso o el texto móvil en tiempo real.
Tabla de Multiplicar: Ingresa un número para ver su tabla con diseño visual.
Mi Galería: Revisa las creaciones que has hecho durante la sesión.
Salir: Cierra el programa y guarda tus avances.
Flujo de Trabajo
El programa funciona en un loop continuo. Esto significa que después de generar un arte, siempre tendrás la opción de regresar al menú principal para probar algo nuevo, a menos que elijas la opción de salida.
Guardado y Persistencia
Automático: Cada vez que generas una pieza de arte, esta se añade a tu lista de sesión.
Archivos: El arte se exporta individualmente a archivos .txt dentro de la carpeta datos/. ¡Puedes abrirlos después con cualquier editor de texto!
Limpieza: Si quieres empezar de cero, usa la opción "Limpiar Galería" dentro del menú de visualización.
